VerySimpleToolTip
=======

Very simple html tooltip with CSS and Javascript wihout lib.

## Getting Started

Insert toolstips.css and toolstips.js.

exemple:
```
<script>
	document.querySelectorAll("a").forEach(function (item) {
		tooltips(item);
	});
</script>
```

### Prerequisites

javascript / css
